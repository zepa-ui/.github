# Zepa

Curated UI components, motion systems, and immersive experiences for modern web applications.

Built for developers who care about details.

---

## What is Zepa?

Zepa is an open-source collection of modern React and Next.js components focused on motion, interaction, and memorable user experiences.

From hero sections to immersive interfaces, every component is designed to be copied, customized, and shipped.

---

## Features

* Curated UI components
* Motion-first interactions
* Scroll-driven experiences
* GSAP and Framer Motion animations
* shadcn-compatible registry
* React and Next.js support
* Open source

---

## Philosophy

Most interfaces work.

Great interfaces feel alive.

Zepa explores the small details that make products memorable:

* Motion
* Interaction
* Depth
* Timing
* Experience
