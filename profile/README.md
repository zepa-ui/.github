## What is Zepa?

Zepa is an open-source collection of modern React and Next.js components focused on motion, interaction, and memorable user experiences.

From hero sections to immersive interfaces, every component is designed to be copied, customized, and shipped.

The web has become faster, but many interfaces have become forgettable. Zepa explores the small details that make products feel alive — movement, timing, depth, interaction, and emotion.

Some components are inspired by existing experiences. Others are built from scratch. Every component is an experiment in making interfaces feel more human.

## Philosophy

Most interfaces work.

Great interfaces stay in people's minds.

Zepa is built around the idea that motion should have meaning, interactions should feel natural, and interfaces should create experiences rather than simply display information.

The goal is not to build the largest component library.

The goal is to build components that developers remember, reuse, and enjoy working with.

## Technology

Built with React, Next.js, TypeScript, Tailwind CSS, GSAP, Framer Motion, and the shadcn ecosystem.

Zepa focuses on modern frontend tooling while keeping components easy to copy, customize, and integrate into real applications.

---

## Current Focus

Today, Zepa is exploring hero sections, scroll-driven experiences, interactive layouts, motion systems, and experimental UI patterns.

The collection continues to grow one component at a time.
